## Community Channels

### Slack

Our Slack community is where you can:

- Get help and support
- Share your Codegen projects
- Connect with other developers
- Stay updated on new features

### GitHub

Codegen is open source and we welcome contributions! On GitHub you can:

- Report issues
- Submit pull requests
- Star the project
- Browse the source code

### Twitter (X)

Follow us on Twitter/X to:

- Get the latest updates
- See community highlights
- Connect with the team
- Share your Codegen wins

## Contributing

We welcome contributions of all kinds! Whether you're fixing a typo in documentation, reporting a bug, or implementing a new feature, we appreciate your help in making Codegen better.

Check out our [Contributing Guide](https://github.com/codegen-sh/codegen-sdk/blob/main/CONTRIBUTING.md) on GitHub to learn how to:

- Set up your development environment
- Submit pull requests
- Report issues
- Contribute to documentation

Together, we can build better tools for code transformation!
